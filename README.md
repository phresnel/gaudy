
gaudy - a C++ header-only color conversion- and computation-library
-------------------------------------------------------------------------------

Features:
* complete <cmath>-implementation for linear RGB
* header only
* no macros (except include guards)


More info later.

Build/Test:

* `scons`
* `scons test`
